# Worldly - Dictionary SPA

A Single Page Application that looks up word definitions, examples, synonyms, and pronunciation using the Free Dictionary API.

##  Features
- Search any English word
- Shows definitions, part of speech, and examples
- Play audio pronunciation
- Lists synonyms
- Clean, responsive UI

##  Tech Used
- HTML5
- CSS3 
- Vanilla JavaScript
- Free Dictionary API: https://api.dictionaryapi.dev

##  How to Use
1. Open the Live Demo or `index.html` in your browser
2. Type a word in the search box
3. Click "Search"
